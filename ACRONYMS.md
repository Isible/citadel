- lit: literal

- tok: token

- arr: array

- vec: vector

- val: value

- AST: Abstract Syntax Tree

- CHR: Citadel high level representation

- CMR: Citadel medium level representation

- CLR: Citadel low level representation

- NYI: Not yet implemented
